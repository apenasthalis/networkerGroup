# Plataforma de Gestão para Grupos de Networking (Networker Group)

## Sumário

O sistema de gestão para grupos de networking é responsável por permitir o cadastro de membros, atualização, listagem, e remoção de grupos, além de gerir informações como preços mensais, categorias, estoques e imagens.

### Solicitar Intenção de Participação
- O administrador pode recusar ou aceitar essa intenção;
- Formulário de cadastro para membros ao serem aceitos no Networker Group;

### Comunicação e Engajamento:
- Área de comunicados e notificações aos membros
- Controle de check-in em reuniões

### Geração de Negócios
- Sistema de indicações e referências de negócios entre membros
- Verificar o estado de cada indicação, se foi enviada ou não.
- Registro de agradecimentos

### Acompanhamento e Perfomance
- Controle de reuniões 1 a 1 entre membros.
- Dashboards de desempenho individual e do grupo.
- Relátorios por Período

### Financeiro
- Módulo de controle de mensalidades (geração, status de pagamento).

### 1. Interface do usuário
- Será uma aplicação web moderna rápida e dinâmica

    **Tecnologia a serem utilizadas:**
    - Frontend em Next.js/React
        - Deverá ser usado o app router nativo do next para a sincronização das rotas.
        - UseQuery da library "tanstack" para consultas;
